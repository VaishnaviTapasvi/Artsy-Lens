# Artsy Lens – Photography Portfolio Website

Artsy Lens is a static photography portfolio website showcasing multiple photography styles such as fashion, wildlife, travel, sports, documentary, wedding, and editorial photography. It contains multiple HTML pages designed to highlight images and information about various photography themes.

### Features

+ Home Page – Overview of the website and categories
+ Photography Categories
  + Fashion Photography
  + Wildlife Photography
  + Travel Photography
  + Sports Photography
  + Editorial Photography
  + Wedding Photography
+ About Us Page
+ Contact / Form Page
+ Terms & Conditions Page
+ 100% static website (HTML-based)
